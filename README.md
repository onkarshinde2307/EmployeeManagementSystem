# Employee Management System

A professional ASP.NET MVC 5 application developed using C#, SQL Server, Entity Framework (Database First), Repository Pattern, and Session Authentication. This project demonstrates CRUD operations with Search, Sorting, Pagination, and Form Validation.

---

## Technologies Used

- ASP.NET MVC 5 (.NET Framework)
- C#
- SQL Server
- Entity Framework 6 (Database First)
- Repository Pattern
- Session Authentication
- Bootstrap 5
- jQuery
- PagedList.Mvc

---

## Features

- User Login
- Session-Based Authentication
- Custom Session Authorization Filter
- Employee CRUD Operations
- Search Employees
- Sort by Name & Salary
- Pagination
- Server & Client Side Validation
- Repository Pattern
- Clean MVC Architecture

## Database Setup

1. Open SQL Server Management Studio (SSMS)
2. Execute:

```
Database/DBEmployee.sql
```

3. Update the Connection String in `Web.config`.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/onkarshinde2307/EmployeeManagementSystem.git
```

2. Open the solution in Visual Studio.

3. Restore NuGet Packages.

4. Execute `Database/DBEmployee.sql`.

5. Update the SQL Server connection string.

6. Build and Run the project.
